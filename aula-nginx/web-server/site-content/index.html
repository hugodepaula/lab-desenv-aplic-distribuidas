Bem-vindo ao Servidor Nginx!

✅ Se você está vendo esta página, o servidor Nginx está funcionando corretamente!

Esta é uma página HTML estática simples que está sendo servida pelo Nginx rodando em um contêiner Docker.

Informações do Servidor:

  • Servidor: Nginx
  • Porta: 8080 (mapeada para a porta 80 do contêiner)
  • Volume: ./site-content mapeado para /usr/share/nginx/html

Este arquivo está localizado em: /usr/share/nginx/html/index.html dentro do contêiner Docker.
